# MVP Features

## Core Features (v1.0)

### Visuals
- [ ] Agent sprites (different colors/types for different roles)
- [ ] Background scene (office/workspace environment)
- [ ] State indicators (idle, working, communicating, error)

### Agent Interaction
- [ ] Agent movement between locations
- [ ] Message bubbles showing communication
- [ ] Connection lines between collaborating agents
- [ ] Task assignment visualization

### UI
- [ ] Agent status panel (list of agents + current state)
- [ ] Task queue display
- [ ] Basic controls (pause/resume simulation)

### Integration
- [ ] Connect to OpenClaw agent system
- [ ] Poll agent states in real-time
- [ ] Display agent communication events
